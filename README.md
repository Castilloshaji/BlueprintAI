# BlueprintAI

BlueprintAI is an AI-powered software architect that transforms software ideas into production-ready technical blueprints.

## Features

- AI Blueprint Generation
- Architecture Design
- Tech Stack Recommendation
- Database Schema Generation
- API Design
- Budget Estimation
- Deployment Planning
- Project Review
- Retrieval-Augmented Generation (RAG)
- LLM-powered Engineering Assistance

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy

### AI
- LangGraph
- LangChain
- Groq/OpenAI
- ChromaDB
